1. BDOVPN
D1, A1, P1
BDOVPN 172.16.0.0/16 25 wfh
C 25 is 5bit
S /32 - /5 = /27
I 4th, 32i
subnet: 172.16.0.32/27
1st  172.16.0.33
last 172.16.0.62
broadcast   172.16.0.63/27
next subnet 172.16.0.64/27

@D1
conf t
vlan 2
name BDOVPN
exit
interface vlan 2
no shut
ip add 172.16.0.33 255.255.255.224
exit
ip dhcp excluded-add 172.16.0.33 172.16.0.37
ip dhcp pool BDOVPN
network 172.16.0.32 255.255.255.224
default-router 172.16.0.33
end

@A1
conf t
int e0/0
no shut
switchport access vlan 2
do sh vlan brief

@P1
conf t
int e0/0
no shut
ip address dhcp
do bp
